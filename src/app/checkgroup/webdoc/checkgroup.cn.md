# Checkgroup 复选组

<div class="used-tiny">

通过`tiCheckgroup`与`tiCheckitem`，实现一组数据的全选功能。

+ 可以不使用全选（`tiCheckgroup`），只在一组数据中做选择。
+ 管理多组交错的数据。

```typescript
import { TiCheckboxModule } from '@opentiny/ng';
```

</div>

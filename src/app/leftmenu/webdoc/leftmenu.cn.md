# Leftmenu 左侧菜单

<div class="used-tiny">

Leftmenu 是提供页面导航的菜单列表。&nbsp;&nbsp;

+ 支持二级菜单。

+ 支持菜单分组。

+ 支持头部、底部区域自定义。

```typescript
import { TiLeftmenuModule } from '@opentiny/ng';
```

</div>

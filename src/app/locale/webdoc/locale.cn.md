# TiLocale 国际化语言

<div class="used-tiny">

Tiny 组件中使用的国际化词条配置方法类。&nbsp;&nbsp;

+ 可通过在启动模块中通过配置具体的国际化语种的方式设置组件的国际化。

```typescript
import { TiLocaleModule } from '@opentiny/ng';
```

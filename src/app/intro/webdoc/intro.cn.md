# Intro 新手引导

<div class="used-tiny">

Intro 是提供服务中内容（入口）新增和变动等指引提醒的组件。&nbsp;&nbsp;

```typescript
import { TiIntroServiceModule } from '@opentiny/ng';
```

需要在项目中（建议在根模块）中导入`BrowserAnimationsModule`。

```typescript
import { BrowserAnimationsModule } from '@angular/platform-browser/animations';
```

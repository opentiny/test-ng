# TagsInput 标签输入

<div class="used-tiny">

TagsInput 是标签输入框组件。&nbsp;&nbsp;

+ 支持输入框输入标签、可以联想选择标签等场景。

```typescript
import { TiTagsInputModule } from '@opentiny/ng';
```

+ 如需使用校验功能，请导入。

```typescript
import { TiValidationModule } from '@opentiny/ng';
```

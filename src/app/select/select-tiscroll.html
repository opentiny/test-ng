描述

tiScroll指令。

在有局部滚动条的容器中使用select，为了防止下拉框在容器滚动时与选择框分离，开发者需要在对应的容器上添加tiScroll指令，使下拉框在容器滚动时消失。
也可以自己监听滚动元素的scroll事件，在回调中触发document下的tiScroll事件，使得容器元素滚动时面板收起

导入

import {{ '{' }} TiScrollModule {{ '}' }} from '@opentiny/ng';

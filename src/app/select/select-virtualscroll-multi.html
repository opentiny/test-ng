描述

Select选择下拉组件, 下拉数据量大的时候可以将 virtual 接口设置为 true 来开启虚拟滚动功能。分组下拉不支持虚拟滚动。

virtual 接口：默认值为 false。

导入

import {{ '{' }} TiSelectModule {{ '}' }} from '@opentiny/ng';

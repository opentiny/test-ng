# Tree 树

<div class="used-tiny">

Tree 是可以完整展现数据的层级关系，并具有展开收起选择等交互功能的组件。&nbsp;&nbsp;

+ 支持单选、多选两种场景。

+ 提供 TiTreeUtil 公共方法，包括增、删、改、查、选中、取消选中、遍历、筛选、获取选中项等操作。

```typescript
import { TiTreeModule } from '@opentiny/ng';
```

# Swiper 轮播

<div class="used-tiny">

播放一组内容。

+ `<ti-swiper>`是轮播的容器；`<ti-swipercard>`是每个轮播内容的容器。
+ 一组轮播由一个`<ti-swiper>`嵌套多个`<ti-swipercard>`组成。

```typescript
import { TiSwiperModule } from '@opentiny/ng';
```

</div>

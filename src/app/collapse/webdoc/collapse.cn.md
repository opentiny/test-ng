# Collapse 折叠面板

<div class="used-tiny">

通过折叠面板收纳内容区域。

```typescript
import { TiCollapseModule } from '@opentiny/ng';
```

需要在项目中（建议在根模块）中导入`BrowserAnimationsModule`。

```typescript
import { BrowserAnimationsModule } from '@angular/platform-browser/animations';
```
</div>

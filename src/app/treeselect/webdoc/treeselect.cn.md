# Treeselect 树选择

<div class="used-tiny">

Treeselect 组件是树选择组件。支持单选、多选等，默认为单选。多选选中后以标签形式呈现，标签可删除。

```typescript
import { TiTreeselectModule } from "@opentiny/ng";
```

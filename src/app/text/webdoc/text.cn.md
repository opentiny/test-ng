# Text 文本框

<div class="used-tiny">

文本输入框。&nbsp;&nbsp;

+ 支持禁用、密码显示、完全清除等场景。

+ 支持通过`tiMask`属性限制用户只能输入数字及设置数字格式，常用于限定身份证、手机号码等格式的输入。

```typescript
import { TiTextModule } from '@opentiny/ng';
```
